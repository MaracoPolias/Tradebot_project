# Security Policy

**tradebot-project**

Security is a critical priority for this project.  
We appreciate responsible and private disclosure of vulnerabilities.

## Supported Versions
The following branches receive security updates:
- `main` (production)  
- `dev` (active development)  

Feature branches receive fixes only when merged into the above.

## Reporting a Vulnerability
Do **not** open public GitHub issues for security problems.

Instead, report privately to:

**githubtradebot@maildrop.cc**

Please include:
- Detailed description of the vulnerability  
- Steps to reproduce  
- Potential impact  
- Suggested fixes (optional)  

You will receive acknowledgment within **48 hours**, and updates within **5 business days** as we investigate.

## Disclosure Process
- Valid reports will be assessed immediately  
- Maintainers will patch the issue on a private branch  
- After validation, a public fix will be released  
- Credit may be given (optional)

Thank you for helping keep tradebot-project secure.
